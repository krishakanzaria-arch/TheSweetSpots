# 🎂 The Sweet Spot - Cake Customization Website

A modern cake ordering website that allows users to customize cakes with a **real-time 3D preview** using Three.js.

---

## 🚀 Features

- 🎨 Customize cake (shape, color, toppings, etc.)
- 🧁 Real-time **3D cake preview**
- 🛒 Interactive UI for cake selection
- 📱 Responsive design
- ⚡ Smooth animations using JavaScript

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Three.js (for 3D rendering)
